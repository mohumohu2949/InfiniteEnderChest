# InfiniteEnderChest
半無限に拡張されたエンダーチェスト！

## 動作バージョン
- 1.20.2

## 前提データパック => OhMyDat
赤石愛様
url : https://github.com/Ai-Akaishi/OhMyDat

## 使い方
エンダーチェストの右列にページをめくるボタンが出現するので、クリックするだけ！

## 注意事項
overworldでないと、プレイサウンドやアイテムを持ったままボタンをクリックしてしまったときに、アイテムが返却されない問題が起こります。
